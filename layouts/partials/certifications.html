Certifications

Azure App Service OSS

Earners of the Azure App Service Open Source Software badge have obtained knowledge in using OSS frameworks within Azure App Service. This includes PHP, WordPress, Node.js, Java, Python, Ruby on Rails, and MySQL. A Specialist has also acquired skills in troubleshooting deployment using Git and BitBucket. They also have knowledge of hosting applications on Linux in App Service, including troubleshooting Docker-related issues.

Azure App Service Development

Earners of the Azure App Service Development badge has achieved a comprehensive skill set in WebJobs, remote debugging, deploying apps to App Service, and gaining insights using Application Insights.
